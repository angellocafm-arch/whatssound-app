## Viral Loops para WhatsSound

### Loop 1: Share Session
```
1. Usuario en sesión escucha canción que le gusta
2. Comparte a WhatsApp/Instagram story
3. Amigo ve y clickea link
4. Amigo entra a sesión (sin registro)
5. Amigo disfruta → se registra
6. Nuevo usuario repite loop
```
**K-factor target:** 0.3 (cada 3 usuarios traen 1 nuevo)

### Loop 2: Golden Boost Notification
```
1. Usuario da Golden Boost a DJ
2. DJ recibe notificación
3. DJ comparte en sus redes "Me han dado Golden Boost en WhatsSound"
4. Followers del DJ descubren la app
5. Nuevos usuarios → nuevos boosts
```
**K-factor target:** 0.2

### Loop 3: Referral Program
```
1. Usuario invita amigo con código
2. Ambos reciben Golden Boost gratis
3. Invitado invita a sus amigos
4. Efecto compuesto
```
**K-factor target:** 0.5+

### Loop 4: DJ Brings Audience
```
1. DJ se registra
2. DJ promociona sesión en sus redes
3. Audiencia del DJ entra a WhatsSound
4. Oyentes se convierten en fans de otros DJs
5. Más DJs quieren registrarse
```
**K-factor target:** Variable (depende del DJ)

## Framework: AARRR para WhatsSound

### Acquisition (Adquisición)
- Organic: SEO, ASO, shares
- Paid: Meta ads, TikTok ads (futuro)
- DJ-driven: DJs traen audiencia

### Activation (Activación)
- Aha moment: Escuchar 1 canción + ver chat activo
- Target: <5 minutos desde registro

### Retention (Retención)
- Push cuando DJ favorito está en vivo
- Weekly digest de DJs seguidos
- Golden Boost refresh reminder

### Referral (Referidos)
- Código de invitación
- Share session
- Share achievements

### Revenue
- Tips, Golden Boost, subs
